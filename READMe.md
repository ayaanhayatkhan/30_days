# 🚀 Ayaan's 30-Day Coding Plan

A modern, interactive **30-Day Coding Roadmap** designed to help learners build strong programming fundamentals through daily practice.

The roadmap covers four programming tracks:

* 💻 C Programming
* ⚡ C++
* 🐍 Python
* 🌐 Web Development (HTML, CSS & JavaScript)

Each day contains carefully selected reading resources and a practical coding assignment, making it easy to stay consistent and measure your progress.

---

## ✨ Features

* 📅 30-Day structured learning roadmap
* 📚 Daily learning resources
* 💡 Practical coding assignments
* ✅ Mark tasks as completed
* 📊 Live progress tracking
* 🔥 Daily streak counter
* 💾 Progress automatically saved using Local Storage
* 🌙 Automatic Light/Dark mode based on system preference
* 📱 Responsive design for desktop and mobile
* 🏆 Mini project milestones
* 🔄 Cross-language challenge indicators

## 📈 Progress Tracking

The application automatically tracks:

* Completed tasks
* Remaining tasks
* Overall completion percentage
* Consecutive full-day streaks

All progress is stored locally in your browser, so you can continue where you left off.
